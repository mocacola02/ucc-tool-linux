# ucc-tool-linux
UCC Tool for Linux. Simple interface to run UCC commands via Wine.

## Features
- UCC Make using custom ini and post-build result message
- Coming soon: `fullbatchexport` support
- Feel free to request additional commands

## Installation
**Requires `Zenity`.**

Download UCC_Tool.sh and set it as executable with `chmod +x UCC_Tool.sh`. The tool can then be opened like a regular executable or run via console.
